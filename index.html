1-¿Cuáles son las principales funciones de un kernel en la arquitectura de un sistema operativo y cómo interactúa con el hardware?

El kernel es el núcleo del sistema operativo y se encarga de gestionar los recursos esenciales del sistema. Sus principales funciones incluyen: la gestión de la memoria, el control de procesos, la gestión de dispositivos y la comunicación entre el hardware y el software. El kernel actúa como intermediario entre las aplicaciones y el hardware, traduciendo las solicitudes de software en instrucciones que el hardware puede entender y ejecutar.

2-¿Cómo se gestionan los procesos y cuál es la importancia de la multitarea en la eficiencia del sistema?

La gestión de procesos implica la creación, planificación, ejecución y terminación de procesos. El sistema operativo asigna recursos como tiempo de CPU y memoria a cada proceso y asegura que se ejecuten sin conflictos. La multitarea permite que varios procesos se ejecuten simultáneamente compartiendo recursos, lo que mejora la eficiencia y la productividad del sistema al aprovechar al máximo la capacidad del hardware y reducir los tiempos de espera para los usuarios.

3-¿Qué medidas de seguridad se pueden implementar a nivel de sistema operativo para proteger una red, especialmente en entornos empresariales?

A nivel de sistema operativo, se pueden implementar varias medidas de seguridad como: la gestión de cuentas y permisos de usuario, la configuración de firewalls, el uso de software antivirus y antimalware, el cifrado de datos, el uso de sistemas de detección de intrusos (IDS), y políticas de actualización automática para mantener el sistema al día. En entornos empresariales, también es fundamental el uso de autenticación multifactor y el monitoreo constante de la actividad de red.
